# Features Section Redesign: Trading Card "Artifacts"

## 1. コンセプト概要（Concept Overview）
Featuresセクションを単なる機能説明ではなく、**「TCG（遊戯王やポケモンカード）のレアカード（アーティファクト）を獲得する」** 体験へと昇華させます。

- **スタッガード（段違い）配置**: カードが空間に浮遊しているように、Y軸（縦方向）をずらして配置します。
- **ホログラフィック・ホバーエフェクト**: カードにマウスを乗せると、カード全体が少し浮き上がり、枠線（ボーダー）に「紫 → 金 → 青」のホイル加工のような光の反射（Glow）が走ります。
- **カードの解剖学（Card Anatomy）**:
  1. **Card Art（画像）**: カードの上半分を占める、美麗なアイテム（アーティファクト）のイラスト。
  2. **Card Type（種類）**: 「[Passive Skill]」「[Legendary Tool]」などのバッジ。
  3. **Card Name（機能名）**: 「Immersion Tracker」など。
  4. **Card Effect（効果・説明）**: 機能の詳細（RPGのスキル説明文風）。

---

## 2. カード用 画像生成コンテキスト（Hero Prop / 象徴的オブジェクト）

背景を複雑な部屋の風景にするのではなく、機能のシンボルとなる**「1つの象徴的なアイテム（Hero Prop）」**にフォーカスします。
ゲーム（ZZZやスタレ）の**インベントリ（装備品）画面**のように、暗い背景に美しい3Dアイテムがポツンと置かれている、シンプルで洗練されたビジュアルを目指します。

### 🃏 カード1：Adaptive Learning（SLAに基づく適応学習）
**【象徴オブジェクト】**: **光るスマートタブレットと抹茶ラテ**
**【エフェクトと質感】**: 深い暗闇（デスクマットの上）に斜めに置かれた、ベゼルレスの薄型タブレット。画面には美しい日本語のコミックが映り、画面から漏れる青白い光が、横に置かれた抹茶ラテのカップの輪郭を照らしている。ミニマルで洗練された3Dセルルック。

### 🃏 カード2：Belajar Bersama Komunitas（コミュニティと共に学ぶ）
**【象徴オブジェクト】**: **ネオンが走るゲーミングマイク**
**【エフェクトと質感】**: 重厚な金属の質感を持つ高級ゲーミングマイク（Shure SM7Bのような形）。マイクのボディに紫とオレンジのRGBライトが鋭く走り、周囲にホログラムのチャットバブル（吹き出し）が数個、フワフワと浮遊している。

### 🃏 カード3：Peta Jalan Terstruktur（構造化されたロードマップ）
**【象徴オブジェクト】**: **ホログラムが浮かぶスマートプランナー**
**【エフェクトと質感】**: 開かれた上質な黒い手帳（またはデジタルプランナー）。そのページの上から、RPGの「スキルツリー」や「路線図」のような金色のホログラムが立体的に浮かび上がっている。手前にはスタイリッシュな万年筆が置かれている。

### 🃏 カード4：Berpikir dalam Jepang（日本語で考える）
**【象徴オブジェクト】**: **ノイズキャンセリング・ヘッドホンとライトノベル**
**【エフェクトと質感】**: 分厚い日本語のライトノベルの上に置かれた、重厚なオーバーイヤー型ヘッドホン。イヤーカップの側面がネオンパープルにうっすらと発光しており、周囲の「翻訳のノイズ」を遮断するような、静寂を感じさせる冷たい光の表現。

### 🃏 カード5：Sentence Miner（文章抽出ツール / Anki連携）
**【象徴オブジェクト】**: **サイバーなポラロイドカメラ（またはスキャナー）**
**【エフェクトと質感】**: レトロフューチャーなデザインの分厚いカメラ。レンズから紫色のスキャンレーザーが放たれ、カメラの口からは「光る日本語の単語カード（フラッシュカード）」がカシャッと出力されている瞬間のエフェクト。

### 🃏 カード6：Content Library（ネイティブ・コンテンツ庫）
**【象徴オブジェクト】**: **積み重なった漫画とレトロなアーケードスティック**
**【エフェクトと質感】**: 暗闇の中に美しく積まれた数冊のコミックとゲームパッケージ。その上に、クリアパーツが光るアーケードコントローラーが置かれている。オブジェクトの隙間から、紫や青のLEDの光が漏れ出し、宝箱のようなワクワク感を放っている。
