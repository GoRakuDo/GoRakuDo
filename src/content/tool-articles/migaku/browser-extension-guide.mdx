# Migaku Browser Extension Guide

## Overview

Migaku Browser Extension adalah tool powerful untuk mining content dari video bahasa Jepang di YouTube, Netflix, dan platform lainnya.

## Installation

### Prerequisites

- Chrome atau Firefox browser
- Migaku account (free atau premium)
- Anki installed (untuk card creation)

### Setup Steps

1. Visit Migaku website
2. Download browser extension
3. Install di browser
4. Login dengan Migaku account
5. Configure initial settings

## Features

### 1. Subtitle Display

**Dual Subtitle Mode**:

- Japanese subtitles (primary)
- Indonesian/English subtitles (secondary)
- Toggle visibility with hotkeys

**Subtitle Customization**:

- Font size dan color
- Position adjustment
- Background transparency
- Reading aids (furigana)

### 2. Word Lookup

**Hover Dictionary**:

- Instant word definitions
- Multiple dictionary sources
- Audio pronunciation
- Example sentences

**Kanji Information**:

- Readings (on/kun)
- Stroke order
- Compounds
- Frequency data

### 3. Sentence Mining

**Quick Mining**:

- Click word to create card
- Auto-capture context sentence
- Include screenshot
- Add audio snippet

**Mining Workflow**:

1. Watch video
2. Hover over unknown word
3. Click "Mine" button
4. Review and export to Anki

## Platform Support

### YouTube

**Features**:

- Auto-load Japanese subs
- Timeline navigation
- Replay controls
- Bookmark system

**Best Practices**:

- Use channels dengan accurate subs
- Mine from native content
- Focus on i+1 sentences

### Netflix

**Features**:

- Dual subtitle support
- Episode bookmarks
- Auto-pause on mine
- Progress tracking

**Limitations**:

- Premium Migaku required
- Limited to Netflix catalog
- Some regions restricted

## Workflow Optimization

### Efficient Mining

**Setup**:

1. Configure hotkeys untuk quick access
2. Set auto-pause duration
3. Enable screenshot capture
4. Configure Anki template

**Process**:

1. Watch video at normal speed
2. Pause only untuk unknown words
3. Quick mine with hotkey
4. Continue watching
5. Review mined cards later

### Card Quality

**Best Cards Include**:

- Target word highlighted
- Full sentence context
- Audio clip (2-3 seconds)
- Screenshot dengan context
- Source information

## Advanced Features

### Subtitle Editing

Edit subtitles untuk better cards:

- Fix timing issues
- Correct mistakes
- Split long sentences
- Merge short phrases

### Custom Exports

Export options:

- Direct to Anki
- CSV for spreadsheet
- JSON for processing
- Text file for review

### Statistics Tracking

Monitor your progress:

- Words mined per day
- Watch time statistics
- Vocabulary growth
- Review accuracy

## Troubleshooting

### Common Issues

**Subtitles tidak muncul**:

- Check video has Japanese subs
- Reload extension
- Clear cache
- Update extension

**Mining tidak bekerja**:

- Verify Anki running
- Check AnkiConnect setup
- Test connection
- Review permissions

**Performance issues**:

- Disable unused features
- Reduce subtitle effects
- Clear mining queue
- Restart browser

## Tips dan Best Practices

### Effective Mining

**Do's**:

- ✅ Mine i+1 sentences
- ✅ Include full context
- ✅ Add audio clips
- ✅ Review regularly

**Don'ts**:

- ❌ Mine too many words
- ❌ Skip context
- ❌ Ignore pronunciation
- ❌ Collect without reviewing

### Content Selection

**Good Sources**:

- Slice-of-life anime
- Japanese dramas
- Documentary content
- Interview shows

**Avoid**:

- Heavily edited content
- Fast-paced action
- Technical jargon (initially)
- Non-standard dialects
