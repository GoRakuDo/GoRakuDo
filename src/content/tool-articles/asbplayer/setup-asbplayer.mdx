## Android: Setup dari Browser Helium

Pada artikel sebelumnya kita sudah menulisa bagaimana cara melalukan setup dasar untuk Sentence Mining di-Android, bagi belum melakakun setup awalnya silahkan diikuti terlebih dahuli di →
[Setup Dasar Sentence Mining di Android](/tutorial/anki/setup-sentence-mining-android).

<div class="account-split">
<ol>
1. Melalui browser Helium, tap icon puzzle (ekstensi) untuk membuka daftar menu ekstensi yang sudah terinstall.
2. Disamping "asbplayer: Language-learning with subtitles" terdapat titik tiga untuk membuka pengaturan ekstensi asbplayer, tap dan pilih Options, maka akan terbuka halaman tab baru pengaturan asbplayer.
3. Karena android memiliki layar yang sempit geser menu bagian atas "anki" "mining" dan seterusnya sampai menemukan bagian "misc". Dan tab bagian "misc" kemudan scroll dalam bagian menunya sampai ke bagian bawah maka kamu bisa menemukan tombol "IMPORT SETTING".
4. "IMPORT SETTING" melalui file backup `asbplayer-settings-2026-7-10-18-20-24.json` yang sudah di-download sebelumnya.
5. Selesai Asbplayer siap digunakan.
</ol>

### Android: Cara menggunakan Asbplayer untuk video player lokal

Untuk Smarphone Android layar yang sempit mungkin akan sedikit sulit, tapi masih tatap cukup ok nyaman jika digunakan dalam mode landscape untuk Sentence Minig. Karena di ekstensi browser Helium android pop-up menu ekstensi terkadang tidak berfungsi. Kalian tetap bisa mengakses Webapp Asbplayer melalui [https://app.asbplayer.dev/](https://app.asbplayer.dev/).

1. Tap "browse" untuk memilih video mp4 / mkv.
2. Dan tap icon folder sekali lagi untuk load memunculkan subtitle bahasa jepang srt / ass.
3. Untuk menggubah "Playback Rate", atau "Subtitle Offset" jika subtitle telat atau kecepatan muncul, tahan bagian nomor timpestampt bagian tengah bawah, dan geser keatas sekali untuk "Sibtitle Offset", dan sekali lagi setelah "Sibtitle Offset" untuk "Playback Rate".
4. Info: Untuk "Subtitle Offset" Masukkan `+1000` plus seribu milidetik untuk memperlambat 1 detik, sbelaiknya `-1000` minus seribu untuk mempercepat 1 detik subtitle muncul. Semua dalam satuan milidetik.

### Android: Cara menggunakan Asbplayer untuk Web Streaming

Untuk Web video streaming seperti YouTube kemingkinan besar terdapat auto subtitle yang bisa langsung kalian pakai, atau kalian bisa menggunakan subtitle manual srt / ass yang kalian miliki. Berikut cara mengunakan ekstensi Asbplayer untuk Web Streaming.

1. Untuk mempermudah silahkan pin ekstensi Asbplayer dengan tap icon puzzle (ekstensi) untuk membuka daftar menu ekstensi yang sudah terinstall.
2. "asbplayer: Language-learning with subtitles" terdapat titik tiga untuk membuka pengaturan ekstensi asbplayer, dan tap "Pin ke ToolBar".
3. Dihalaman video dari YouTube, tap icon ekstensi Asbplayer hingga terbuka popup "Select Subtitles", dibagian "Subtitle Track 1" pilih "Bahasa Jepang (Auto Generate Subtitle)" atau Open Files jika kalian memiliki subtitle manual srt / ass.
4. Jika muncul popup "enable audio recording", silahkan tap sekali lagi icon ekstensi Asbplayer yang ada di toolbar.

## Komputer: Setup dari Browser Chrome

Untuk komputer selama kalian menggunakan browser yang berbasais Chromium pengaturan setup kurang lebih akan tetap sama. Pastikan kalian sudah menginstall ekstensi Asbplayer dari [Google Chrome WebStore](https://chromewebstore.google.com/detail/asbplayer-language-learni/hkledmpjpaehamkiehglnbelcpdflcab).

![Asbplayer Chrome extension settings import](/content/tools-asbplayer/setup-asbplayer/asbplayer_settings_import.gif)

1. Jika ekstensi asbplayer sudah terinstall, buka daftar ekstensi dari icon puzzle, disamping "asbplayer: Language-learning with subtitles" terdapat titik tiga klik untuk membukan opsi ekstensi, dan klik "Options" maka akan tebuka page halaman pengaturan dari Asbplayer.
2. Dalam halaman pengaturan ekstensi Asbplayer pilih menu "MISC / Lain-Lain", dan scroll ke plaing bawah dari pengaturan tersebut.
3. Terdapat tombol "IMPORT SETTING", klik dan import backup `asbplayer-settings-2026-7-10-18-20-24.json` yang sudah kamu download di awal.
4. Selesai, ekstensi asbplayer ini siap kamu pakai.

### Komputer: Cara menggunakan Asbplayer untuk video player

![Asbplayer webapp video player walkthrough](/content/tools-asbplayer/setup-asbplayer/asbplayer_webapp_video_player.gif)

1. Klik icon ekstensi Asbplayer, dalam popup menu pilih "App" untuk membuka Webplayer dari Asbplayer.
2. Klik browse di tengah untuk memilih video mp4 / mkv.
3. Untuk menambahkan subtile srt / ass dari komputer kalian, arakan kursor sedikit ke kiri atas, dan klik icon folder untuk menambahkan subtile.
4. Kiri bawah dekat tombol play / pause terdapat baian untuk mengature "Subtitle Offset" berfungsi jika substilenya telat muncul atau kecepatan muncul. Masukkan `+1000` plus seribu milidetik untuk memperlambat 1 detik, sbelaiknya `-1000` minus seribu untuk mempercepat 1 detik subtitle muncul. Semua dalam satuan milidetik.
5. Di sebelah "Subtitle Offset", terdapat bagian "Playback Rate" untuk mengatur kecepatan video diputar.

### Komputer: Cara menggunakan Asbplayer untuk Web Streaming

![Asbplayer YouTube web streaming walkthrough](/content/tools-asbplayer/setup-asbplayer/asbplayer_youtube_webstream.gif)

Sebagai contoh kita menggunakan YouTube dengan Asbplayer.
1. Klik icon ekstensi Asbplayer, dalam popup menu pilih "SIDE PANEL" untuk menampilkan subtitle Sidepanel dari Asbplayer.
2. Di bagian sidepanel klik tombol "LOAD SUBTITLE", jika muncul popup "Multiple Video Elemnts Detected" pilih yang ada gambar thumbnalnya, dan ok.
3. Selanjutnya dalam popup "Select Subtitle" terdapat bagian "Subtitle Track 1" klik dan jika kalian memiliki subtitle manual ass / srt terdapat opsi "Open Files", atau bisa menggunakan "Bahasa Jepang (Auto Generate Subtitle)".

## Pengaturan lainnya

![Asbplayer enbale video capture option](/content/tools-asbplayer/setup-asbplayer/asbplayer_video_capture_support.webp)

Jika Deck Anki kalian mendukung format media video untuk ditampilkan dalam kartu Anki bisa menggubah bagian pengaturan ekstensi Asbplayer → MINING → Image capture → Capture format.   Dibagian ini kalian dapat memilih "Video clip (loacal video only)" maka setiap gambar di karu anki kalian akan bergerak sebagai video. Atau "Screenshot" hanya gambar screenshot.

![Asbplayer enable take screenshot](/content/tools-asbplayer/setup-asbplayer/asbplayer_take_screenshot_option.webp)

Dan jika kalian menggunakan pengaturan sendiri, seacara pabrik ekstensi Asbplayer tidak akan melakukan Screenshot. Kamu bisa  menyalakan opsi "Take screenshot when mining" melalui pengaturan ekstensi Asbplayer → STREAMING VIDEO → Mining → "Take screenshot when mining".
