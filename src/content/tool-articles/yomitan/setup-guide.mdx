# Setup Guide Lengkap Yomitan

## Prerequisites

Sebelum memulai, pastikan Anda memiliki:

- Browser Chrome atau Firefox
- Anki terinstal (optional, untuk integrasi)
- AnkiConnect add-on (jika menggunakan Anki)

## Instalasi Yomitan

### Step 1: Install Extension

1. Buka Chrome Web Store atau Firefox Add-ons
2. Cari "Yomitan"
3. Klik "Add to Chrome" atau "Add to Firefox"
4. Izinkan permissions yang diminta

### Step 2: Install Dictionaries

Yomitan memerlukan dictionary files:

1. Klik icon Yomitan di toolbar
2. Go to Settings
3. Navigate to "Dictionaries"
4. Click "Import"
5. Download dan import dictionary files:
   - JMdict (English)
   - JMdict (Indonesian) jika tersedia
   - Kanjidic
   - KANJIDIC2 for kanji information

## Konfigurasi Dasar

### Dictionary Settings

Configure dictionary priority:

1. English definitions: Priority 1
2. Indonesian definitions: Priority 2
3. Kanji dictionary: Priority 3

### Display Settings

Optimize popup display:

- **Font size**: 14-16px
- **Max entries**: 10
- **Audio**: Enable auto-play
- **Popup position**: Follow cursor

### Keyboard Shortcuts

Default shortcuts:

- **Alt + Insert**: Toggle Yomitan
- **Alt + A**: Add to Anki
- **Alt + N**: Next definition
- **Alt + P**: Previous definition

## Anki Integration

### Setup AnkiConnect

1. Install AnkiConnect dalam Anki
2. Start Anki
3. Buka Yomitan Settings
4. Go to "Anki" section
5. Enable "Connect to Anki"

### Card Template

Configure card format:

**Front**:

```
{expression}
{sentence}
```

**Back**:

```
{reading}
{glossary}
{audio}
```

## Advanced Settings

### Scanning Options

- **Scan length**: 1-10 characters
- **Scan delay**: 50ms
- **Scan modifier key**: None (always on)

### Audio Configuration

Setup audio sources:

1. JapanesePod101
2. Jpod101 Alternate
3. Jisho.org

## Tips Penggunaan

### Efficient Reading

1. Hover over words untuk instant lookup
2. Press Alt+A untuk add to Anki
3. Click arrow untuk melihat definisi lain
4. Use keyboard shortcuts untuk navigasi cepat

### Mining Strategy

Best practices untuk sentence mining:

- Mine i+1 sentences (1 unknown word)
- Include full sentence for context
- Add audio untuk pronunciation
- Tag dengan source (anime, manga, etc.)

## Troubleshooting

### Common Issues

**Yomitan tidak muncul**:

- Check extension is enabled
- Reload page
- Check permissions

**Dictionary tidak load**:

- Re-import dictionary files
- Check storage space
- Clear browser cache

**Anki integration tidak bekerja**:

- Pastikan Anki running
- Check AnkiConnect installed
- Verify port settings (default 8765)

## Kesimpulan

Dengan setup yang proper, Yomitan akan menjadi tool yang sangat powerful untuk immersion reading. Take time untuk configure sesuai preferensi Anda untuk pengalaman terbaik.

**Next Steps**:
