# YomitanとAnkiの連携方法

Yomitanで調べた単語をAnkiカードとして自動生成する方法を解説します。

## 基本的な連携設定

### AnkiConnectのインストール

- AnkiConnectプラグインの導入
- 接続設定の確認
- ポート設定の調整

### Yomitanの設定

- Anki連携の有効化
- カードテンプレートの設定
- フィールドマッピングの調整

## カード自動生成の設定

### デフォルトテンプレート

- フロント側の設定
- バック側の設定
- スタイリングの調整

### カスタムテンプレート

- HTMLテンプレートの作成
- CSSスタイルの適用
- JavaScript機能の追加

## 高度な連携機能

### バッチ処理

- 複数単語の一括登録
- 重複チェック機能
- エラーハンドリング

### 同期機能

- リアルタイム同期
- バックアップ機能
- 復元機能

## トラブルシューティング

### 接続エラー

- ポート設定の確認
- ファイアウォール設定
- プロキシ設定

### カード生成エラー

- テンプレートの確認
- フィールド名の一致
- 権限設定の確認
