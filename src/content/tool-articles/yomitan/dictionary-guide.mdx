# Panduan Dictionary Yomitan

## Dictionary Types

### 1. Word Dictionaries

**JMdict (English)**

- Most comprehensive Japanese-English dictionary
- Include part of speech information
- Multiple definitions per word
- Example sentences

**JMdict (Multilingual)**

- Supports multiple languages
- Good for non-English speakers
- Translations dalam berbagai bahasa

### 2. Kanji Dictionaries

**KANJIDIC2**

- Comprehensive kanji information
- Stroke order
- Reading variants
- Usage frequency

**Kanji Frequency**

- Kanji sorted by frequency
- Useful untuk prioritas pembelajaran
- Based on news corpus

### 3. Pitch Accent Dictionaries

**Kanjium Pitch Accent**

- Tokyo-standard pitch accent
- Essential untuk pronunciation
- Visual pitch patterns

## Installation Process

### Download Sources

Recommended sources untuk dictionary files:

1. **Official Yomitan Dictionary Repository**
   - Verified dan tested
   - Regular updates
   - Comprehensive collection

2. **Community Dictionaries**
   - Specialized dictionaries
   - Custom formats
   - Additional languages

### Import Steps

1. Download dictionary (.zip file)
2. Buka Yomitan Settings
3. Go to "Dictionaries" tab
4. Click "Import"
5. Select downloaded file
6. Wait for import completion

## Dictionary Priority

### Ordering Strategy

Set priority based on usefulness:

**Tier 1 (Highest Priority)**:

1. JMdict (English)
2. JMdict (Indonesian) if available

**Tier 2 (High Priority)**: 3. KANJIDIC2 4. Pitch Accent Dictionary

**Tier 3 (Medium Priority)**: 5. Frequency Dictionary 6. Name Dictionary

**Tier 4 (Low Priority)**: 7. Specialized dictionaries 8. Slang dictionaries

## Optimization Tips

### Storage Management

Dictionary files can be large:

- JMdict: ~70MB
- KANJIDIC: ~30MB
- Total recommended: ~200MB

**Optimization**:

- Only install dictionaries you need
- Periodically remove unused ones
- Use compressed formats when available

### Display Optimization

Configure for best reading experience:

**Max Entries**: 5-10

- Too many = overwhelming
- Too few = miss important info

**Max Definitions**: 3-5 per entry

- Focus on most common meanings
- Reduce clutter

## Troubleshooting

### Common Issues

**Dictionary tidak load**:

- Re-import the file
- Check file integrity
- Verify format compatibility

**Definitions tidak muncul**:

- Check dictionary enabled
- Verify priority settings
- Look for conflicts

**Slow performance**:

- Reduce max entries
- Disable unused dictionaries
- Clear cache
