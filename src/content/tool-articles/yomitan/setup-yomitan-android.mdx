Tutorial lanjutan dari ["Setup Dasar Sentence Mining di Android"](/tutorial/anki/setup-sentence-mining-android), kita akan melukan setup Ekstensi Yomitan di Browser Android Helium yang sudah kita install sebelumnya. Setup ini tidak akan ribet karena kita telah menyiapkan Backup siap pakai untuk kalian yang baru mulai Immerison / Belajar bahasa Jepang secara Otodidak.

## Inport Pengaturan dan Kamus dari Backup

Untuk mempermudah kalian melakuakn pengaturan ekstensi Yomitan, kita telah menyiapkan semua pengaturannya secara lengkap dan siap pakai. Terdapat "Kamus JMdict" dan pengaturan lainnya untuk pemula yang baru mulai Sentence Mining dengan NoteType DenChou.

Silahkan download terlebih dahulu ke-dua Backup Yomitan ini:
- Backup Kamus: [yomitan-dictionaries-2026-07-06-09-48-52.json](https://transfer.it/t/8vBoRWxjRLCZ)
- Backup Settings: [yomitan-settings-2026-07-06-09-48-37.json](https://transfer.it/t/Ek11UTxjzRRI)

### Langkah-langkah melakukan Pengaturan dari Backup

<div class="account-split">
<div>

1. Tap Icon "Puzzle" di bagian atas samping URL bar dari Browser Helium untuk membuka daftar Ekstensi yang sudah terinstall.
2. Jika sudah terinstall dengan benar maka terdapat Ekstensi "Yomitan Popup Dictionary", dan tap titik tiga disampingnya untuk membuka "Options".
3. Setelah tap "Options", maka akan terbuka tab halaman pengaturan Yomitan.
4. Scroll sampai paling bawah kalian akan menemukan bagian "Backup".
5. Pertama silahkan tap tombol "Import Dictionary Collection" dan pilih file `yomitan-dictionaries-2026-07-06-09-48-52.json` yang sudah kalian download sebelumnya. Dan tunggu hingga proses Import selesai maka akan muncul teks hijau "Done Importing".
6. Selanjutnya di ikuti dengan "Import Settings" dan pilih file `yomitan-settings-2026-07-06-09-48-37.json`. Dan tunggu hingga proses Import selesai maka akan muncul teks hijau "Done Importing".
7. Restart Browser Helium melalui pengaturan Android Aplikasi, temukan browser Helium, dan Force Stop. Dan buka kembali Browser Helium.

</div>
<img src="/content/tools-yomitan/setup-yomitan-android/yomitan_backup_import.gif" alt="Yomitan backup import step by step" width="540" height="1230" loading="lazy" />
</div>

Selesai. Ekstensi Yomitan kalian siap langsung di-pakai.

## Cara menggunakan Ekstensi Yomitan

Kamu bisa test dengan membuka website apapun di Browser Helium. Jika terdapat bahasa Jepang tap tahan sepersekian detik saja (jangan ditahan terlalu lama), maka akan mucul pop-up dari kamus Yomitan ini.
