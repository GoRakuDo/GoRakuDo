Artikel tutorial ini kita akan melakukan setup awal keperluan untuk Sentence Mining di Android. Dan perlu diketahui setup dan pengaturan Sentence Mining bisa berubah dari waktu ke waktu dikarenakan versi masing-masing aplikasi dan tools terus berubah.

Pastikan kamu sudah menginstal AnkiDroid dari Play Store. Kemudian download dan install NoteType **DenChou** melalui tutorial yang sudah tertulis → [Cara Setup DenChou](/project/denchou/setup_overview).

## Browser Android untuk Sentence Mining

Kalian bisa juga menggunakan Firefox Android, tapi untuk kompatibilitas dan performa yang lebih baik kita merekomendasikan menggunakan Browser Helium versi Android yang berdasarkan Chromium Desktop. Sehingga memiliki support ekstensi Google Chrome.

![Setup browser sentence mining pake android-helium-browser](/content/tools-anki/setup-sentence-mining-android/android-helium-browser-release.png)

Download APK-nya melalui Repository GitHub [android-helium-browser](https://github.com/jqssun/android-helium-browser/releases). Per tanggal 2 Juli 2026 kami merekomendasikan menggunakan versi v149.0.7827.197, dan terdapat 2 jenis `armeabi-v7a` dan `arm64-v8a` silahkan tanya ke AI ChatGPT atau Gemini yang mana yang cocok dengan tipe HP Android kamu. Download dan install apk browsernya.

## Install & Setup AnkiConnect Android

AnkiConnect merupakan aplikasi yang akan menghubungkan ekstensi kamus Yomitan dari browser Helium ke aplikasi AnkiDroid kalian. Jadi pastikan kedua aplikasi ini "AnkiDroid" dan "AnkiConnect" tidak masuk dalam daftar power saving di HP Android kalian. Karena kebanyakan HP Android seperti Xiaomi dan teman-temannya selalu memaksa semua aplikasi berhenti bekerja di latar belakang.

Cara mematikan Power Saving-nya gimana? Tanya Google, AI atau ChatGPT karena kemungkinan setiap merek berbeda-beda.

<div class="account-split">
<div>
1. Download dari Github Release [AnkiconnectAndroid](https://github.com/KamWithK/AnkiconnectAndroid/releases). Dan install apk-nya.
2. Allow / Izinkan semua popup permintaan akses dari AnkiConnect
3. Buka Setting melalui icon roda di kanan atas, tap dan pilih  "Access Overlay Permissions" dan Allow berikan Izin untuk "Display over other apps".
4. Selesai. Setiap kalian mau Sentence Mining silahkan tap "START SERVICES". Dan pastikan sudah menonaktifkan Battery Saver untuk AnkiDroid dan AnkiConnect.
</div>
