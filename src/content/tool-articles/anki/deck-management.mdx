# Strategi Manajemen Deck Anki

## Struktur Deck yang Efektif

### Hierarchical Organization

Organisir deck Anda secara hierarkis:

```
Japanese
├── Vocabulary
│   ├── Core 2K
│   ├── JLPT N5
│   └── Daily Immersion
├── Kanji
│   ├── RTK
│   └── Frequency Order
└── Grammar
    ├── Genki I
    └── Genki II
```

### Flat Structure

Alternatif untuk yang lebih sederhana:

- Japanese - Vocabulary
- Japanese - Kanji
- Japanese - Grammar
- Japanese - Sentences

## Tag System

### Tag Categories

Gunakan tag untuk kategorisasi lebih detail:

- **Source**: `anime`, `manga`, `news`, `textbook`
- **Level**: `beginner`, `intermediate`, `advanced`
- **Type**: `vocabulary`, `kanji`, `grammar`
- **Status**: `review`, `mastered`, `difficult`

### Tag Hierarchy

Contoh penggunaan tag hierarkis:

```
jlpt::n5::vocabulary
jlpt::n4::grammar
anime::slice-of-life::vocabulary
```

## Deck Options

### New Cards Settings

- **New cards/day**: 10-20 untuk pemula
- **Graduating interval**: 1 hari
- **Easy interval**: 4 hari

### Reviews Settings

- **Maximum reviews/day**: 200
- **Easy bonus**: 130%
- **Interval modifier**: 100%

## Import/Export Strategy

### Importing Shared Decks

1. Download dari AnkiWeb
2. Review struktur deck
3. Merge dengan deck existing jika perlu
4. Sesuaikan tag dan options

### Exporting for Backup

- Export dengan media included
- Gunakan `.apkg` format
- Simpan di multiple locations

## Maintenance Routine

### Daily

- Review semua due cards
- Add 10-20 new cards
- Update problematic cards

### Weekly

- Check database
- Review statistics
- Adjust deck options if needed

### Monthly

- Full backup
- Clean up unused media
- Reorganize tags if needed

## Best Practices

### Dos

- ✅ Konsisten dengan naming convention
- ✅ Gunakan tag secara liberal
- ✅ Regular backup
- ✅ Review dan refine cards

### Don'ts

- ❌ Terlalu banyak subdeck
- ❌ Duplicate cards
- ❌ Ignore statistics
- ❌ Add terlalu banyak new cards sekaligus

## Kesimpulan

Manajemen deck yang baik adalah kunci untuk pembelajaran jangka panjang yang efektif. Dengan struktur yang terorganisir, Anda dapat fokus pada pembelajaran tanpa distraksi.
