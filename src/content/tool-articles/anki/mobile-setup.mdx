# Setup Anki di Mobile

## Platform Overview

### AnkiDroid (Android)

**Keunggulan**:

- Gratis dan open source
- Full feature parity dengan desktop
- Highly customizable
- Support untuk add-ons tertentu

**Download**: Google Play Store

### AnkiMobile (iOS)

**Keunggulan**:

- Official Anki app
- Optimized untuk iOS
- Smooth performance
- iCloud sync support

**Download**: App Store ($24.99)

## Installation Guide

### Android Setup

1. Buka Google Play Store
2. Search "AnkiDroid"
3. Install aplikasi
4. Buka dan configure permissions

### iOS Setup

1. Buka App Store
2. Search "AnkiMobile Flashcards"
3. Purchase ($24.99 one-time)
4. Install aplikasi

## Sync Configuration

### Create AnkiWeb Account

Jika belum punya:

1. Visit ankiweb.net
2. Click "Sign Up"
3. Enter email dan password
4. Verify email address

### Setup Sync di Mobile

**Android**:

1. Open AnkiDroid
2. Tap menu â†’ Settings
3. Tap "AnkiWeb account"
4. Enter credentials
5. Tap "Sync"

**iOS**:

1. Open AnkiMobile
2. Tap Settings icon
3. Tap "Synchronization"
4. Enter AnkiWeb credentials
5. Tap "Sync"

## First Sync

### From Desktop to Mobile

1. Ensure latest review di desktop
2. Sync desktop Anki
3. Open mobile app
4. Tap sync button
5. Wait for download

### Conflict Resolution

Jika ada conflict:

- Choose "Download from AnkiWeb" (first sync)
- Choose "Upload to AnkiWeb" (if mobile has latest)

## Troubleshooting

### Sync Issues

**Problem**: Sync fails

**Solutions**:

- Check internet connection
- Verify AnkiWeb credentials
- Force close and reopen app
- Clear app cache

### Media Not Syncing

**Problem**: Audio/images missing

**Solutions**:

- Ensure media synced on desktop
- Check storage space
- Re-sync media
- Verify file formats

### Performance Problems

**Problem**: App slow or crashes

**Solutions**:

- Clear cache
- Reduce deck size
- Update app
- Restart device
