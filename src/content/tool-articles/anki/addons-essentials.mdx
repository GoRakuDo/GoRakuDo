# Add-ons Anki yang Wajib Dimiliki

## Essential Add-ons

### 1. Japanese Support

**Fungsi**: Menambahkan support lengkap untuk bahasa Jepang

**Fitur**:

- Furigana generation otomatis
- Pitch accent display
- Audio pronunciation
- Kanji information

**Install**: Code `3918629684`

### 2. AnkiConnect

**Fungsi**: API untuk integrasi dengan tools eksternal

**Fitur**:

- Integrasi dengan Yomitan
- Import otomatis dari browser
- Sync dengan aplikasi mobile
- Custom automation

**Install**: Code `2055492159`

### 3. Advanced Browser

**Fungsi**: Enhance search dan filter capabilities

**Fitur**:

- Custom columns
- Advanced search syntax
- Bulk editing
- Statistics display

**Install**: Code `874215009`

## Productivity Add-ons

### 4. Review Heatmap

Visualisasi aktivitas review Anda dalam bentuk heatmap untuk menjaga motivasi dan konsistensi.

**Install**: Code `1771074083`

### 5. Image Occlusion Enhanced

Untuk membuat kartu dengan bagian gambar yang di-occlude, sangat berguna untuk belajar kanji dan diagram.

**Install**: Code `1111933094`

### 6. Speed Focus Mode

Menambahkan timer otomatis untuk review, membantu Anda tetap fokus dan efisien.

**Install**: Code `1046608507`

## Learning Enhancement

### 7. Morphman

Analisis morfologi teks Jepang untuk pembuatan kartu yang lebih cerdas.

**Fitur**:

- Difficulty analysis
- Frequency tracking
- Smart card generation
- i+1 sentence mining

### 8. AJATT Japanese

Template kartu yang dioptimalkan untuk metode AJATT (All Japanese All The Time).

**Fitur**:

- Audio on front
- Context sentences
- Minimal design
- Fast review

## Customization Add-ons

### 9. Custom Background Image

Personalisasi Anki dengan background image favorit Anda.

### 10. Night Mode

Dark theme untuk review malam hari, mengurangi eye strain.

**Install**: Code `1496166067`

## Installation Guide

### Cara Install Add-ons

1. Buka Anki
2. Tools → Add-ons → Get Add-ons
3. Masukkan add-on code
4. Restart Anki

### Troubleshooting

Jika add-on tidak bekerja:

1. Check compatibility dengan versi Anki Anda
2. Disable add-ons lain yang conflict
3. Restart Anki
4. Check forum Anki untuk solusi

## Configuration Tips

### Optimal Settings

Setelah install, configure add-ons:

1. Japanese Support: Enable furigana auto-generation
2. AnkiConnect: Allow browser connections
3. Review Heatmap: Set calendar view
4. Speed Focus Mode: Set 10-15 seconds per card

## Kesimpulan

Add-ons ini akan mengubah Anki menjadi powerhouse untuk pembelajaran bahasa Jepang. Mulai dengan essential add-ons, lalu tambahkan yang lain sesuai kebutuhan.

**Rekomendasi Instalasi**:

1. Japanese Support (wajib)
2. AnkiConnect (wajib)
3. Review Heatmap (sangat recommended)
4. Advanced Browser (recommended)
5. Lainnya sesuai preferensi
