# Troubleshooting Anki: Solusi untuk Masalah Umum

## Sync Problems

### Issue 1: "Sync Failed" Error

**Symptoms**:

- Sync button shows error
- Changes tidak tersimpan
- Timeout messages

**Solutions**:

1. **Check Internet Connection**

   ```
   - Verify connection stable
   - Try different network
   - Disable VPN temporarily
   ```

2. **Force Full Sync**

   ```
   Tools â†’ Preferences â†’ Network
   - Click "Force changes in one direction"
   - Choose upload or download
   - Confirm sync
   ```

3. **Check AnkiWeb Status**
   - Visit ankiweb.net/status
   - Wait if maintenance
   - Try again later

### Issue 2: Sync Conflict

**Symptoms**:

- "Modified on both devices" message
- Don't know which version to keep

**Solutions**:

**Method 1: Choose Newer**

- If desktop has latest: Upload
- If mobile has latest: Download
- Backup before choosing

**Method 2: Manual Merge**

1. Export deck from both devices
2. Compare changes
3. Keep desktop version
4. Manually add mobile changes
