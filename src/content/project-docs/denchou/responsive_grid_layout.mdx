# Tata Letak Kisi Responsif

Kuasai sepenuhnya tata letak kartumu. Kamu bisa mengatur **Word** (Kata) dan **Picture** (Gambar) secara berdampingan (**Horizontal**) atau menumpuknya satu sama lain (**Vertikal**). Kamu juga bisa membalik urutannya untuk menempatkan gambar di kiri, kanan, atas, atau bawah, secara terpisah untuk ukuran layar yang berbeda.
