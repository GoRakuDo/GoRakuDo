# Catatan Kalimat

<video src="/page_images/project/DenChou/docs/assets/sentence_l1_note.mp4" controls autoplay loop muted playsinline></video>

Bagian ini sebelumnya adalah `SentenceTranslation`, jika kamu pindah dari Senren, field `sentenceTranslation` sudah tidak dipakai lagi di DenChou. Ganti nama field itu secara manual menjadi `SentenceL1Note` di note type Anki kalian. 

Bagian ini untuk menampilkan catatan kalian ketika sentence mining (jika kalian menulis bidang `SentenceL1Note`).
