# Persiapan: Anki

## Cara Mengurutkan Kartu Anki berdasarkan Frekuensi (pake Add-on AutoReorder)

Kalian bisa secara otomatis mengurutkan kartu yang baru ditambahkan berdasarkan frekuensi dengan langkah berikut:

<div class="tqa-split">
<ol>
1.  Instal add-on [AutoReorder](https://ankiweb.net/shared/info/757527607) (`757527607`).
2.  Buka Tools -> AutoReorder Options...
3.  Tempelkan konfigurasi berikut:

    ```json
    {
        "search_to_sort": "deck:DenChou is:new", // Sesuaikan nama deck jika diperlukan
        "shift_existing": true,
        "sort_field": "freqSort",
        "sort_reverse": false
    }
    ```

4.  Pastikan bidang (field) `freqSort` pada tipe kartu DenChou sudah dicentang bagian "Sort by this field in the browser" di dalam dialog Fields milik Anki.
</ol>

## Cara Install Font (Lokal diKomputer Kalian)

Agar tampilan tetap konsisten, terutama jika menggunakan Anki di berbagai perangkat, kamu bisa menginstal font-nya secara lokal di dalam Anki (lewat Komputer):

1.  Unduh file fonta dari folder [Fonts](https://github.com/GoRakuDo/DenChou/tree/main/Fonts) di repositori DenChou.
    *   [`_Hiragino Kaku Gothic ProN W3.otf`](https://github.com/GoRakuDo/DenChou/raw/refs/heads/main/Fonts/_Hiragino%20Kaku%20Gothic%20ProN%20W3.otf)
    *   [`_KleeOne-SemiBold.ttf`](https://github.com/GoRakuDo/DenChou/raw/refs/heads/main/Fonts/_KleeOne-SemiBold.ttf)
    *   [`_NotoSansJP.ttf`](https://github.com/GoRakuDo/DenChou/raw/refs/heads/main/Fonts/_NotoSansJP.ttf) (Alternatif untuk Hiragino)
    *   [`_NotoSerifJP.ttf`](https://github.com/GoRakuDo/DenChou/raw/refs/heads/main/Fonts/_NotoSerifJP.ttf) (Alternatif untuk Klee)
2.  Letakkan file fonta yang diunduh (diawali dengan `_`) ke dalam folder `collection.media` di profil Anki kamu. Jalurnya biasanya seperti ini: 
    *   `C:\Users\[NamaPenggunaKamu]\AppData\Roaming\Anki2\[NamaProfilKamu]\collection.media` (Windows) 
    *   `~/Library/Application Support/Anki2/[NamaProfilKamu]/collection.media` (macOS)
