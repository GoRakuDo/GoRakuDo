# Kanji Hover

Fitur ini diteruskan dari pendahulu DenChou, yaitu SenRen. Kalian bisa arahkan kursor (hover) di atas kanji apa pun untuk menampilkan hingga lima kosakata lainnya yang menggunakan kanji yang sama di koleksi deck-mu. Jika ada lebih banyak kartu, opsi “**Show More**” (Tampilkan Lebih Banyak) akan muncul. Mengklik kosakata terkait di dalam tooltip akan membuka kartu yang bersangkutan di Daftar Kartu (Card Browser) Anki.

## Kustomisasi

Kamu bisa mengontrol bagaimana kanji, penurunan pitch (pitch downstep), dan kata-kata terkait diurutkan di dalam tooltip melalui variabel di bagian `/* PITCH ACCENT SETTINGS */` pada CSS:
```css
--kanji-hover-pitch-colors:    true; /* "false" untuk menonaktifkan pewarnaan pitch downstep di Kanji Hover */
--kanji-hover-pitch-highlight: true; /* "false" agar warna pitch berhenti menimpa urutan kata + kalimat di Kanji Hover */
```

Kedua opsi aktif secara default. Untuk menonaktifkan pewarnaan pitch sepenuhnya, atur kedua variabel ke `false`.

> **Hanya untuk PC**: Fitur ini saat ini hanya berfungsi pada Anki versi desktop.
