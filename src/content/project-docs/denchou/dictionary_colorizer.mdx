# Pewarna Kamus

Secara otomatis menerapkan warna unik untuk setiap entri kamus, sehingga memudahkan untuk mengidentifikasi sumber definisi sekilas saja. Warna diterapkan pada bingkai (border), judul, penghitung (counter), dan tautan.

**Cara Penggunaan:**

Kamu bisa menyesuaikan fitur ini dengan mengedit **Styling** di templat kartu Anki kamu. Cari bagian `/* DICTIONARY & GLOSSARY STYLING */`.

1.  **Aktifkan Pewarna**  
    Cari `--dictionary-colorizer`. Atur ke `1` untuk mengaktifkan fitur ini.
    ```css
    --dictionary-colorizer: 1;
    ```

2.  **Latar Belakang**  
    Cari `--dict-bg-enabled`.
    *   Atur ke `1`: Memberikan warna latar belakang halus pada kotak definisi kamus (Default).
    *   Atur ke `0`: Menjaga latar belakang tetap netral.
    ```css
    --dict-bg-enabled: 1;
    ```

3.  **Latar Belakang Kartu Penuh**  
    Cari `--dict-back-bg`.
    *   Atur ke `1`: Warna kamus akan memenuhi **seluruh latar belakang kartu**.
    *   Atur ke `0`: Menjaga latar belakang kartu utama tetap netral.
    ```css
    --dict-back-bg: 1;
    ```

> **Sesuaikan Warnamu**
>
> Tidak suka dengan warna default? Kamu bisa mengubahnya!
> Gulir ke bawah ke bagian `/* Dict Colors */` di dalam CSS. Kamu bisa mengubah kode warna untuk mode Terang (Light) dan Gelap (Dark) untuk kamus tertentu mana pun.
