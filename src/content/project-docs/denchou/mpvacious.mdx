# Persiapan: Mpvacious

Gunakan mpvacious untuk membuat kartu langsung dari [mpv media player](https://mpv.io/).

1.  Pastikan kamu telah menginstal [mpvacious](https://github.com/Ajatt-Tools/mpvacious) dengan benar untuk mpv.
2.  Download file konfigurasi DenChou: [subs2srs.conf](/page_images/project/DenChou/resources/subs2srs.conf) (Klik kanan → Simpan Tautan Sebagai... / Save Link As...)
3.  Letakkan file `subs2srs.conf` di dalam direktori `script-opts` milik mpv.
    *   Lokasi AppData Default: `%AppData%\mpv\script-opts\`
    *   Buat folder `script-opts` jika belum ada.
4.  Unduh file custom [subs2srs.lua](/page_images/project/DenChou/resources/subs2srs.lua) (Klik kanan → Simpan Tautan Sebagai... / Save Link As...)
5.  Letakkan file `subs2srs.lua` di dalam `%AppData%\mpv\scripts\subs2srs\` dan pilih "replace the file in destination" (ganti file di tujuan) ketika diminta.

Konfigurasi ini memetakan bidang-bidang (fields) mpvacious ke bidang-bidang tipe kartu DenChou.
