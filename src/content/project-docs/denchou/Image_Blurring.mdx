# Kartu NSFW

Tambahkan tag `NSFW` pada kartu mana pun, dan gambar yang terkait akan secara otomatis diburamkan saat ditampilkan. Mengeklik gambar akan mengungkapnya.

![Pemblururan Gambar](/page_images/project/DenChou/docs/assets/images/Image_Blurring.gif)

Selain itu, kartu yang ditandai sebagai `NSFW` dapat membuat audio kalimatnya dibisukan (mute) pada awalnya (kamu masih bisa memutarnya menggunakan tombol putar ulang atau pintasan).
> Dinonaktifkan secara default.

Opsi-opsi ini dapat dikonfigurasi melalui variabel di bagian `/* Default States (true/false) */` pada CSS:
```css
--blur-nsfw-picture: true;  /* Buramkan gambar pada kartu NSFW */
--mute-nsfw-audio:   false; /* Bisukan audio kalimat pada kartu NSFW */
```

Atur masing-masing opsi ke `true` untuk mengaktifkannya atau `false` untuk menonaktifkannya.
