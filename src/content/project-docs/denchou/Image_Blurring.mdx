# Kartu NSFW

Tambahkan tag `NSFW` pada kartu apapun sehingga gambar yang ada dalam kartu akan secara otomatis diburamkan saat ditampilkan. Mengeklik gambar akan menampilkannya.

<video src="/page_images/project/DenChou/docs/assets/Image_Blurring.mp4" controls autoplay loop muted playsinline></video>

Selain itu, kartu yang ditandai dengan tag `NSFW` akan membuat audio menjadi dibisukan (mute). Kalian masih bisa memutarnya menggunakan tombol putar ulang atau pintasan.

Pengaturan ini dapat dikonfigurasi melalui variabel di bagian `/* Default States (true/false) */` pada CSS:
```css
--blur-nsfw-picture: true;  /* Buramkan gambar pada kartu NSFW */
--mute-nsfw-audio:   false; /* Bisukan audio kalimat pada kartu NSFW */
```

Atur masing-masing opsi ke `true` untuk mengaktifkannya atau `false` untuk menonaktifkannya.
