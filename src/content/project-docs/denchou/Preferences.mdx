# Pengaturan Preferensi

Pengaturan Menu DenChou yang memungkinkan kamu menyesuaikan tipe kartu langsung di dalam peninjau (reviewer), tanpa memerlukan pengetahuan CSS!

![Pratinjau Pengaturan](/page_images/project/DenChou/docs/assets/preferences_preview.mp4)

### Cara Kerja

1. **Pratinjau Real-Time** Semua penyesuaian yang dilakukan di menu Preferensi akan langsung diterapkan secara langsung setelah menekan "Save to Anki".
2. **Sistem Perubahan Prefrensi** Ada dua jenis:
    *   **Sementara (Temporary):** Pengaturan disimpan secara lokal selama sesi kamu berlangsung. Pengaturan tersebut tetap aktif selama kamu terus melakukan review kartu.
    *   **Permanen (Permanent):** Untuk menerapkan perubahanmu secara permanen, klik **"Save to Anki"**. Ini secara langsung akan menyimpan prefrensi yang kamu rubah, dan langsung diperbarui ke model kartu.
 
### Sistem Preset
*  Preset disimpan di file khusus: (`_denchou_presets.js`) yang terletak di folder `collection.media` Anki kamu.
*  File preset dikelola oleh pengguna. Dengan preset jika kamu update template kartu di versi mendatang tidak akan menimpa preset milikmu. Preset tersebut tidak akan tersentuh kecuali jika kamu sendiri secara eksplisit membuat, mengganti nama, atau menghapus preset melalui menu pengaturan.
*  Karena file preset disimpan di folder media, preset akan tersinkronkan di berbagai perangkat. (Tapi hanya dapat diedit di PC.)

> **Persyaratan**
>
> Untuk menyimpan prefrensi-mu add-on **AnkiConnect** diperlukan, agar fungsi **"Save to Anki"** dapat menyimpan preset.
> Di perangkat mobile (AnkiDroid), menu pengaturan tetap berfungsi, tetapi setiap perubahan bersifat sementara (tidak tersimpan permanent) dan hanya bertahan untuk satu sesi review.
