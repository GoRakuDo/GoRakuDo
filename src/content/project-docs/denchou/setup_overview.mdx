# Panduan Umum

Panduan ini membahas cara mempersiapkan Yomitan dan Anki agar berfungsi optimal dengan Template Kartu Anki "DenChou".

## Instalasi Pertama Kali
1.  Download file `.apkg` dari dari dari tombol Download yang ada di-atas, atau melalui [Github Release Terbaru](https://github.com/GoRakuDo/DenChou/releases/latest).
2.  Buka Anki / AnkiDroid.
3.  Klik **File -> Import** (atau tekan **Ctrl + Shift + I**) dan pilih file `DenChou.apkg` yang telah kamu download. Untuk Android tap titik 3 yang ada dikanan atas, dan Inport `DenChou.apkg` yang telah di-download.

> **Update Pembaharuan**
>
> Jika kamu update memperbarui dari versi sebelumnya, pastikan untuk mengaktifkan opsi Merge note types saat mengimpor.

| Pembaruan |  |
| :--- | :--- |
| **Merge note types** | 🔵 **Enabled** (On) |
| **Update notes** | Always |
| **Update note types** | Always |

## Memperbarui Tanpa MengDownload APKG
1. Buka **Browse** di Anki.
2. Pilih Tipe Kartu **DenChou**.
3. Klik **Cards**.
4. Ganti konten pada bagian **Front Template**, **Back Template**, dan **Styling** dengan file yang disediakan berikut:
    - [`front.html`](https://github.com/GoRakuDo/DenChou/raw/refs/heads/main/Template/front.html)
    - [`back.html`](https://github.com/GoRakuDo/DenChou/raw/refs/heads/main/Template/back.html)
    - [`styling.css`](https://github.com/GoRakuDo/DenChou/raw/refs/heads/main/Template/styling.css)
5. **Simpan** setelah melakukan perubahan.
