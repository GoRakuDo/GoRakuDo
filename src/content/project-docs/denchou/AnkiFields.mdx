# Konfigurasi: AnkiFields

Konfigurasi Anki fields untuk Sentence Mining / membuat kartu secara otomatis mengisi bidang (field) DenChou.

## Konfigurasi Bidang (Fields Configuration):

Dalam Preferences → Anki:

| Bidang (Field) | Nilai Terpilih (Selected Value) |
| :--- | :--- |
| **word** | Primary Spelling |
| **reading** | Readings |
| **sentence** | Sentence |
| **sentenceFurigana** | Nothing |
| **sentenceTranslation** | Nothing |
| **sentenceCard** | Nothing |
| **audioCard** | Nothing |
| **notes** | Nothing |
| **selectionText** | Selected Definitions |
| **definition** | Definitions |
| **wordAudio** | Audio |
| **sentenceAudio** | Nothing |
| **picture** | Image (Clipboard) |
| **glossary** | Nothing |
| **pitchAccents** | Nothing |
| **pitchPositions** | Pitch Accents (Numeric) |
| **pitchCategories** | Nothing |
| **frequencies** | Frequencies |
| **freqSort** | Frequency (Harmonic Mean) |
| **miscInfo** | Nothing |
| **dictionaryPreference** | Nothing |
