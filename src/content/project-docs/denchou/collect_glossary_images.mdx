# Kumpulkan Gambar Glosarium

Menampilkan gambar dari kamus glosarium di dalam wadah gambar (picture container). Kamu bisa menggunakan tombol kiri dan kanan pada gambar untuk menjelajahinya tanpa perlu mencari langsung di kamus. Jika kartu tidak memiliki gambar tetapi kamus glosarium memilikinya, gambar tersebut akan ditampilkan secara otomatis di dalam wadah gambar.

### Diaktifkan
    ![terang](/page_images/project/DenChou/docs/assets/images/collect_glossary_images_enabled.png)

### Dinonaktifkan
    ![gelap](/page_images/project/DenChou/docs/assets/images/collect_glossary_images_disabled.png)

Fitur ini mungkin menyebabkan masalah dengan kamus yang memerlukan CSS tambahan di Anki. Jika kamu menemui masalah, disarankan untuk menonaktifkannya melalui variabel di bagian `/* Default States (true/false) */` pada CSS:
```css
--collect-glossary-images: true; /* Mengizinkan gambar glosarium dikumpulkan ke dalam wadah gambar */
```

Fitur ini aktif secara default. Ubah menjadi `false` untuk menonaktifkannya.
