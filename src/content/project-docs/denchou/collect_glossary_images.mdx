# Gambar dari Kamus Glosarium

Fitur ini diteruskan dari pendahulu DenChou, yaitu Senren. Kalian dapat menggunakan kembali, atau menampilkan gambar dari kamus glosarium untuk wadah gambar (picture container). Jika kartu tidak memiliki gambar tetapi kamus glosarium memilikinya, gambar tersebut akan ditampilkan secara otomatis di dalam wadah gambar.

    ![Gambar dari Kamus Glosarium](/page_images/project/DenChou/docs/assets/images/collect_glossary_images.gif)

Fitur ini dapat menyebabkan masalah dengan kamus yang memerlukan CSS tambahan di Anki. Jika kamu menemui masalah, disarankan untuk menonaktifkannya melalui variabel di bagian `/* Default States (true/false) */` pada CSS:
```css
--collect-glossary-images: true; /* Mengizinkan gambar glosarium dikumpulkan ke dalam wadah gambar */
```

Fitur ini aktif secara default. Ubah menjadi `false` untuk menonaktifkannya.
