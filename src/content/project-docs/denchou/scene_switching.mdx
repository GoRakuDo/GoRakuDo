# Peralihan Adegan

Beralih di antara "Adegan" (Scenes) imersif di mana Kalimat (Sentences), Audio, Gambar (Images), dan Informasi Tambahan (Misc Info) disinkronkan.

![tipe:video](/page_images/project/DenChou/docs/assets/Scene_Switching_Preview.mp4)

- **Navigasi Terkelompok (Grouped Navigation):** Mendukung pembagian kalimat menjadi segmen-segmen menggunakan tag `<span class="group">...</span>` di dalam bidang (field) Anki.
- Untuk menggunakan fitur ini, perbarui bidang `sentence` dan `sentenceFurigana` pada pengaturan "**Configure Anki flashcards…**" di Yomitan:

    <img width="627" height="161" alt="image" src="https://github.com/user-attachments/assets/12506bd0-d03a-456a-90de-b86c4118d61f" />

  - Atur `sentence` ke: `<span class="group">{cloze-prefix}<span class="highlight">{cloze-body}</span>{cloze-suffix}</span>`
  - Atur `sentenceFurigana` ke: `<span class="group">{sentence-furigana}</span>`
    
    Pastikan **Append** (Tambahkan) diaktifkan agar kamu bisa menambahkan kalimat baru alih-alih menimpa kalimat pertama.
    Untuk melakukan ini, aktifkan “**Allow Overwriting**” (Izinkan Penimpaan):

    <img width="671" height="204" alt="image" src="https://github.com/user-attachments/assets/2f7a705f-ab9d-4f0d-a9b2-ad3a1e0a49e2" />
    
-   `sentenceEng` dan `miscInfo` juga didukung. Namun, kamu perlu mengedit bidang tersebut secara manual untuk mengelompokkannya dengan benar pada setiap adegan menggunakan `<span class="group">...</span>`.
    - Kamu bisa memilih adegan mana yang memuat teks tersebut dengan memberi nomor pada grupnya, contohnya `<span class="group2">...</span>`. Ini akan melewati adegan pertama dan mengelompokkannya ke adegan kedua.
    - Jika kamu memiliki beberapa adegan tetapi hanya satu `sentenceEng`, teks tersebut hanya akan muncul di adegan pertama kecuali jika kamu menetapkannya ke grup tertentu seperti contoh di atas.
    > Tidak perlu memberi nomor pada grup jika jumlah terjemahan sama dengan jumlah kalimat.

  - Berikut adalah tampilan di dalam HTML bidang tersebut:
  ```html
  <span class="group2">The future seems bleak...</span><span class="group3">Looks like I have a hard path ahead.</span>
  ```
  ![tipe:video](https://github.com/user-attachments/assets/ccc24115-a1f8-4c67-a8d1-cb7985887a8f)

  - Contoh format lainnya:
  ```html
  <span class="group">Something you wanna try?</span><span class="group">Why don't we test it out?</span><span class="group">Don't you want to try it out?</span>
  ```
  ```html
  <span class="group">My Dress-Up Darling • Season 2, Episode 5 • 0:17:53</span><span class="group">16bit Sensation: Another Layer • Season 1, Episode 13 • 0:16:21</span><span class="group">The Irregular at Magic High School • Season 1, Episode 11 • 0:21:58</span>
  ```
  ![Scene Switching](/page_images/project/DenChou/docs/assets/images/scene_switching_preview.gif)

- **Deteksi Otomatis:** Secara otomatis mendeteksi jika sebuah kartu memiliki beberapa file audio. Jika ditemukan, fitur ini akan mengaktifkan mode Navigasi Adegan; jika tidak, kartu akan kembali ke perilaku standar secara default. Kalimat yang banyak tidak akan dibagi jika kartu tidak berisi setidaknya 2 file audio.
- **Pintasan Putar Ulang Audio Kustom:** Secara default, Anki memutar setiap file audio saat kamu menekan **R**. Pintasan kustom ini hanya memutar audio untuk adegan yang aktif. Tombol default-nya adalah **W**, kamu bisa mengubahnya melalui variabel di bagian `/* CUSTOM SHORTCUTS */` pada CSS:
    ```css
    --scene-replay-shortcut-key: w; /* Atur tombol untuk memutar ulang adegan aktif */
    ```

*   **Pintasan Navigasi Adegan**

    **Depan:** Gunakan tombol **Panah Kiri / Kanan**.
    
    **Belakang:** Gunakan tombol **Ctrl + Panah Kiri / Kanan**
    > Ctrl diperlukan di sini untuk mencegah konflik dengan navigasi Kamus.
