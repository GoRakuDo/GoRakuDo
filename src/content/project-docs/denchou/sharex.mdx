# Persiapan ShareX

Gunakan ShareX untuk mengambil tangkapan layar (screenshot) dan cuplikan audio dengan cepat untuk kartu Anki kamu.

1.  Unduh dan instal [ShareX](https://getsharex.com/).
2.  Unduh file pengaturan ShareX DenChou: [DenChou-ShareX-Settings.sxb](/page_images/project/DenChou/resources/DenChou-ShareX-Settings.sxb) (Klik kanan -> Simpan Tautan Sebagai... / Save Link As...)
3.  Impor pengaturan: Di ShareX -> Application Settings -> Settings (Tab) -> Import -> pilih file `.sxb` yang telah diunduh.
4.  **Perbarui Jalur Pintasan (Hotkey Paths):** Kamu HARUS memperbarui jalur penyimpanan agar mengarah ke folder media Anki *milikmu*.
    *   Buka ShareX -> Hotkey settings...
    *   Untuk masing-masing hotkey **Screenshot**, **Screenshot-NSFW**, dan **Audio**:
        *   Klik tugas tersebut untuk menyorotnya, lalu klik **Edit**.
        *   Centang **Override screenshots folder**.
        *   Ubah jalurnya ke folder `collection.media` dari profil Anki *milikmu* (misalnya, `C:\Users\Administrator\AppData\Roaming\Anki2\NamaProfilAnkiKamu\collection.media`).
        *   Klik **Ok**.

## Pengaturan Perekaman Audio

Untuk mengaktifkan perekaman audio:

1.  Di ShareX -> Hotkey settings... -> Edit hotkey **Audio**.
2.  Buka Screen Recorder dan klik **Screen recording options...**.
3.  Klik **Install recorder devices**.
4.  Atur **Video source** ke `none`.
5.  Atur **Audio source** ke `virtual-audio-capturer`.

## Hotkey Default

*   `Alt + C`: Ambil tangkapan layar -> Simpan ke media Anki -> Tambahkan tag `<img>` ke bidang `picture` pada kartu terakhir.
*   `Alt + N`: Ambil tangkapan layar -> Simpan ke media Anki -> Tambahkan tag `<img>` ke bidang `picture` pada kartu terakhir -> Tambahkan tag `NSFW` ke kartu terakhir.
*   `Alt + V`: Rekam audio -> Simpan ke media Anki -> Tambahkan tag `[sound:]` ke bidang `sentenceAudio` pada kartu terakhir.
*   `PrtSc`: Tangkapan layar standar -> Simpan ke folder ShareX default -> Salin gambar ke papan klip (clipboard).

> **Urutan Penggunaan**
>
> Gunakan hotkey ini **SETELAH** membuat kartu di Anki. ShareX menargetkan kartu yang *terakhir diubah* (last modified).
