# Persiapan Yomipv

Gunakan Yomipv untuk membuat kartu tanpa perlu keluar dari MPV atau mengganggu proses imersi (immersion) kamu.

![tipe:video](/page_images/project/DenChou/docs/assets/yomipv_demo.mp4)

## Persyaratan

- **[MPV](https://mpv.io/)** (0.33.0 atau yang lebih baru)
- **[FFmpeg](https://ffmpeg.org/)** (Diperlukan untuk ekstraksi media, akan dialihkan ke enkoder internal MPV jika tidak ditemukan)
- **[Anki](https://apps.ankiweb.net/)** dengan **[AnkiConnect](https://ankiweb.net/shared/info/2055492159)**
- **[Yomitan](https://yomitan.wiki/)** dan **[Yomitan Api](https://github.com/yomidevs/yomitan-api)**
- **curl** (Sudah terpasang di sebagian besar sistem, digunakan untuk permintaan API)
- **[Node.js](https://nodejs.org/)** (Hanya diperlukan jika menginstal dari sumber (source) atau berkontribusi)

## Instalasi

### Disarankan
1. Unduh [versi terbaru untuk OS kamu](https://github.com/BrenoAqua/Yomipv/releases/latest)
2. Ekstrak kontennya langsung ke dalam direktori MPV kamu:
    - Windows: `%APPDATA%/mpv/`
    - Linux/macOS: `~/.config/mpv/`

### Alternatif (Memerlukan Node.js)
1. **Klon repositori** ke direktori MPV kamu dan instal dependensi **(pastikan kamu telah menginstal Node.js)**:
   - Windows: `%APPDATA%/mpv/`
     ```
     git clone https://github.com/BrenoAqua/Yomipv && xcopy /e /i /y Yomipv . && rd /s /q Yomipv && cd scripts\yomipv\lookup-app && npm install
     ```
   
   - Linux/macOS: `~/.config/mpv/`
     ```
     git clone https://github.com/BrenoAqua/Yomipv && cp -rn Yomipv/* . && rm -rf Yomipv && cd scripts/yomipv/lookup-app && npm install
     ```

## Penggunaan

**Konfigurasi Pengaturan**:
   - Buka `script-opts/yomipv.conf` dan perbarui nama deck/tipe kartu Anki kamu serta pemetaan bidangnya (field mappings)

**Layanan Eksternal**:
   - Pastikan Anki berjalan dengan AnkiConnect yang aktif
   - Pastikan Yomitan Api berjalan dan browser tempat ekstensi Yomitan terinstal dalam keadaan terbuka, serta kamu telah menginstal kamus-kamus

### Alur Kerja Dasar

1. Buka video dengan terjemahan (subtitle) bahasa Jepang di MPV
2. Tekan **`c`** or **gerakkan mouse setelah periode diam** (jika `selector_trigger_on_mouse_move` diaktifkan) untuk mengaktifkan pemilih kata (word selector)
3. Navigasi dengan **mengarahkan mouse (hover)** atau **tombol panah** untuk memilih kata
4. Tekan **`Enter`**, **`c`**, atau **klik kiri** untuk membuat kartu Anki

### Fitur Lanjutan

- **Mode Tambah (Append Mode) (`Shift+C`)**: Pilih beberapa baris subtitle sebelum mengekspor
  - Tekan `Shift+C` untuk masuk ke mode tambah, `c` untuk memulai pemilih kata, atau `Shift+C` lagi untuk membatalkannya

- **Substitusi & Pewarnaan Subtitle (`S`)**: 
  - Tekan **`S`** untuk beralih antara subtitle bawaan MPV dan token berwarna Yomipv
  - Aktifkan `substitute_mpv_subtitles` di `yomipv.conf` agar fitur ini langsung aktif saat memulai
  - Kata-kata diwarnai berdasarkan metadata kartu Anki mereka:
    - **Status**: Baru (New), Sedang Dipelajari (Learning), Ditinjau (Review), Ditangguhkan (Suspended)
    - **Interval**: Mencerminkan seberapa baik sebuah kata diketahui (memengaruhi tingkat kecerahan warna)
    - **Persyaratan**: Tekan **`B`** untuk membangun/menyinkronkan database Anki lokal terlebih dahulu sebelum status-status ini dapat ditampilkan untuk koleksi kartumu saat ini
  - **Umpan Balik Instan (Instant Feedback)**: Ketika kamu membuat kartu, kata tersebut akan segera ditambahkan ke database lokal dan disorot (merah) pada subtitle saat ini

- **Subtitle Sekunder (Secondary Subtitle)**:
  - Memilih subtitle sekunder secara otomatis berdasarkan bahasa pilihan
  - Konfigurasikan `secondary_sub_lang` di `yomipv.conf`

- **Navigasi Tingkat Mora (Mora-level Navigation)**:
  - Ketika `selector_mora_hover` diaktifkan, mengarahkan kursor (hover) di atas kata akan mempersempit pencarian untuk dimulai dari mora di bawah kursor kamu, alih-alih dari kata lengkapnya
  - **`s`**: Beralih navigasi keyboard tingkat mora (tombol kiri/kanan akan berpindah per mora, bukan per kata)

- **Aplikasi Pencarian (Lookup App) (`Ctrl+c`)**: Membuka jendela pop-up yang didukung oleh kamus Yomitan kamu, menampilkan definisi, aksen pitch, dan frekuensi
  - **Klik kanan** pada kata di pemilih untuk mengunci pencarian
  - **Klik mora mana saja** di bagian tajuk (header) untuk mempersempit pencarian ke sub-kata
  - **Klik kanan pada tajuk (header)** untuk kembali ke kata sebelumnya
  - **Aksen Pitch**: Beralih opsi `lookup_show_pitch_accents` di `yomipv.conf`
  - **Frekuensi**: Beralih opsi `lookup_show_frequencies` di `yomipv.conf`
  - Lihat [docs/lookup-app.md](docs/lookup-app.md) untuk detail lengkapnya

- **Mode Persisten (Persistent Mode) (`v`)**: 
  - Beralih mode persisten untuk mengekspor beberapa kata dari satu pemilihan subtitle tanpa menutup pemilih
  - Tekan **`v`** untuk beralih; warna pilihan akan berubah untuk menunjukkan bahwa mode ini aktif
  - Mengonfirmasi pilihan akan mengekspor kartu tetapi membiarkan pemilih tetap terbuka untuk pilihan berikutnya

- **Pemicu Otomatis Pemilih (Auto-Trigger Selector) (`z`)**:
  - Membuka pemilih secara otomatis dengan menggerakkan mouse setelah beberapa saat diam.
  - Aktifkan `selector_trigger_on_mouse_move` dan sesuaikan `selector_trigger_mouse_idle_time` di `yomipv.conf`

- **Waktu Manual (Manual Timing)**:
  - **`q`** / **`w`**: Atur waktu mulai/selesai kustom untuk ekstraksi audio dan gambar
  - Waktu mulai atau selesai yang tidak disetel secara default akan mengikuti batas subtitle saat membuka pemilih
  - **`e`**: Hapus waktu manual

- **Panel Riwayat (History Panel) (`a`)**: Beralih panel riwayat subtitle
  - Klik pada baris sebelumnya/berikutnya untuk memperluas baris subtitle (saat pemilih terbuka)
  - Lompat ke stempel waktu (timestamp) subtitle tertentu dengan mengekliknya (saat pemilih ditutup)
  - **`Alt+LEFT`** / **`Alt+RIGHT`**: Lompat ke subtitle sebelumnya/berikutnya

- **Pelacakan AniList (AniList Tracking) (`Ctrl+A`)**: Mengintegrasikan AniList dengan Yomipv, memungkinkan pembaruan kemajuan episode secara otomatis.
  - Lihat [docs/anilist_tracking.md](docs/anilist_tracking.md) untuk persiapan dan detail lengkapnya

- **Pembaruan Otomatis (Auto-Updater) (`U`)**: Menjaga Yomipv tetap diperbarui ke versi terbaru
  - Tekan **`U`** di MPV untuk memicu pembaruan, atau:
    - Di Windows: Jalankan **`yomipv-updater.bat`** secara langsung
    - Di Linux / macOS: Jalankan **`yomipv-updater.sh`** secara langsung
  - Pilih antara rilis resmi terbaru atau kode sumber (source) terbaru (cabang/branch main)
  - Secara otomatis mempertahankan konfigurasi pengguna di `script-opts/yomipv.conf`
  - Mengunduh biner khusus platform untuk Lookup App
  - (Hanya mode Source) Memperbarui dependensi untuk Lookup App (memerlukan Node.js)
  - Memerlukan hak istimewa administrator (administrator privileges) untuk menjalankan skrip PowerShell di Windows

## Troubleshooting

### Windows
- Pastikan kebijakan eksekusi (execution policy) PowerShell mengizinkan skrip
- Periksa apakah curl tersedia di `C:\Windows\System32\curl.exe`

### Linux
- Pastikan `curl`, `unzip`, `grep`, and `sed` telah terinstal
- Pastikan skrip pembaru memiliki izin eksekusi: `chmod +x yomipv-updater.sh`
- Untuk lookup app, pastikan file biner di `scripts/yomipv/` memiliki izin eksekusi

### macOS
- Pastikan `curl`, `unzip`, `grep`, and `sed` telah terinstal
- Pastikan skrip pembaru memiliki izin eksekusi: `chmod +x yomipv-updater.sh`
- Jika macOS memblokir Lookup App dengan peringatan Gatekeeper, jalankan: `xattr -cr scripts/yomipv/YomipvLookup.app`
- Direktori konfigurasi MPV: `~/.config/mpv/` (buat folder tersebut jika belum ada)
