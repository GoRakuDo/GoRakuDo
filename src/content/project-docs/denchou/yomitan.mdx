# Persiapan: Yomitan

Berikut adalah Konfigurasi Yomitan untuk melakukan Sentence Mining, agar bagian bidang (field) DenChou ter-isi otomatis saat membuat kartu.

## Konfigurasi Bidang (Fields Configuration)

Dalam Yomitan Settings → Anki → Configure Anki Flashcards:

| Field                 | Value                                                                       | Note         |
|-----------------------|-----------------------------------------------------------------------------|--------------|
|  word                 | `{expression}`                                                              |              |
|  reading              | `{reading}`                                                                 |              |
|  sentence             | `<span class="group">{cloze-prefix}<span class="highlight">{cloze-body}</span>{cloze-suffix}</span>`   |  `{cloze-prefix}<b>{cloze-body}</b>{cloze-suffix}` atau `{cloze-prefix}<span class="highlight">{cloze-body}</span>{cloze-suffix}` juga dapat berfungsi jika kamu tidak berniat menggunakan fitur peralihan adegan (scene switching). Baik tag `<b>` maupun `<span class="highlight">` didukung untuk menandai kata target. ||
|  sentenceFurigana     | `<span class="group">{sentence-furigana}</span>`                           | Sebagai alternatif, gunakan `<span class="group">{sentence-furigana-plain}</span>`. Sama seperti `sentence`, kamu bisa menghapus bagian luar `<span>` jika tidak berniat menggunakan fitur peralihan adegan. |
|  SentenceL1Note       |                                                                             | Biarkan kosong. |
|  sentenceCard         |                                                                             | Biarkan kosong. |
|  audioCard            |                                                                             | Biarkan kosong. |
|  notes                |                                                                             | Biarkan kosong. | 
|  selectionText        | `{popup-selection-text}`                                                    | Teks terpilih dari halaman pencarian atau pop-up memiliki prioritas di atas kamus di bidang `definition`. Jika tidak ada teks yang dipilih, ini tidak akan berpengaruh apa-apa. |
|  definition           | `{single-glossary-jmdict}`                                                  | Atur kamus utamamu di sini, atau biarkan kosong. Kamus utama dapat diatur di bidang `dictionaryPreference`. |
|  wordAudio            | `{audio}`                                                                   |              |
|  sentenceAudio        |                                                                             | Biarkan kosong. |
|  picture              |                                                                             | Biarkan kosong. |
|  glossary             | `{glossary}`                                                                | Ada dua cara untuk menggunakan bidang ini. Pertama adalah dengan mengaturnya ke `{glossary}`, ini akan mengekstrak definisi dari semua kamusmu. <br /> Kedua adalah dengan mengaturnya ke kamus tertentu (misalnya `{single-glossary-新選国語辞典-第十版}{single-glossary-デジタル大辞泉}{single-glossary-漢検漢字辞典-第二版}`), yang hanya akan menyertakan kamus yang dipilih. <br /><br /> Ini memerlukan opsi "Group term-reading pairs" atau "Group related terms" untuk diaktifkan di Result Display [(lihat di bawah)](#result-display-setting). |
|  pitchAccents         | `{pitch-accents}`                                                           | Memerlukan kamus aksen pitch. |
|  pitchPositions       | `{pitch-accent-positions}`                                                  | Memerlukan kamus aksen pitch. |
|  pitchCategories      | `{pitch-accent-categories}`                                                 | Memerlukan kamus aksen pitch. |
|  frequencies          | `{frequencies}`                                                             | Memerlukan kamus frekuensi. |
|  freqSort             | `{frequency-harmonic-rank}`                                                 | Digunakan untuk mengurutkan kartu baru. Memerlukan kamus frekuensi. |
|  miscInfo             | `{document-title}`                                                          | Mengambil judul halaman web sumber. Biarkan kosong jika menggunakan mpvacious atau alat serupa yang mengisi bagian ini. |
|  dictionaryPreference |                                                                             | Masukkan `glossary` untuk menampilkan glosarium terlebih dahulu, biarkan kosong untuk menampilkan `definition` terlebih dahulu, atau masukkan nama kamus untuk memprioritaskannya (misalnya, `新選国語辞典　第十版`). Jika Jitendex adalah kamus utamamu tetapi kamu menginginkan kamus lain untuk kartu tertentu, edit bidang ini di **Anki** dan ini akan diprioritaskan tanpa menggunakan Teks Pilihan. <br /><br /> Gunakan [add-on Batch Editing](https://ankiweb.net/shared/info/291119185) untuk memperbarui kartu yang sudah ada. |

## Pengaturan Tampilan Hasil (Result Display Setting)

<div class="tqa-split">
<div>

Penting! Jika kamu menggunakan beberapa kamus, agar bidang `glossary` berfungsi dengan benar perlu mengaktifkan pengelompokan di Yomitan:

Buka **Yomitan Settings** → **Appearance** → **Result Display** dan pilih salah satu:
*   **Group term-reading pairs** ATAU
*   **Group related terms**
